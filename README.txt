# Osadnicy

Opis projektu:
Jest to implementacja w C++ gry planszowej "Osadnicy z Catanu". Zasady zostaną zaczerpnięte z oryginalnej gry. Opis zasad w oryginalnej grze: http://www.nest.republika.pl/catan1.html

Konfiguracja projektu:
Instrukcja dodawania biblioteki SFML do Visual C++: http://cpp0x.pl/kursy/Kurs-SFML-2-x-C++/Instalacja-i-konfiguracja/Konfiguracja-SFML-2-0-Visual-C++/463

Zastosowane wzorce projektowe:
Kompozyt (konstrukcja pól na planszy)
Budowniczy (konstrukcja planszy)
MVC (oddzielenie grafiki obiektów nad planszy od ich działania)

Lokalizacja:
Diagramy UML - folder Diagramy_UML
Pliki programu i kod źródłowy - folder Osadnicy
Biblioteka SFML - folder SFML-2.3.2
